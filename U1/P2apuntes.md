


a) __Componentes básicos de un sistema informático necesario para el desarrollo de software.__

Un sistema informático para el desarrollo de software requiere hardware (procesador, memoria, almacenamiento), software (sistema operativo, herramientas de desarrollo) y periféricos (teclado, monitor). El hardware proporciona la capacidad de procesamiento, mientras que el software incluye entornos de desarrollo y compiladores.  Estos componentes trabajan juntos para facilitar la creación, prueba y depuración de software.

b) __Concepto de software. Historia y orígenes del desarrollo de software.__

El software es el conjunto de programas, datos e instrucciones que permiten a una computadora realizar tareas específicas. Sus orígenes se remontan a los años 40 y 50, con las primeras máquinas programables como ENIAC. En los 60, surgieron lenguajes de alto nivel como COBOL y FORTRAN, facilitando el desarrollo. La evolución continuó con la aparición de sistemas operativos y metodologías de desarrollo estructurado, marcando el inicio de la industria del software moderna.

c) __Concepto de aplicación informática. Fases de desarrollo de una aplicación informática.__

Una aplicación informática es un programa diseñado para realizar tareas específicas, como procesar texto o gestionar bases de datos. Su desarrollo sigue fases como análisis de requisitos, diseño, implementación, pruebas y mantenimiento. En el análisis, se definen las necesidades del usuario. El diseño establece la arquitectura del sistema. La implementación es la codificación, seguida de pruebas para detectar errores. Finalmente, el mantenimiento asegura su funcionamiento continuo.

d) __Lenguajes de programación: conceptos de código fuente, código objeto y código ejecutable.__

El código fuente es el texto escrito por el programador en un lenguaje de alto nivel. El código objeto es la traducción del código fuente a lenguaje máquina, pero aún no ejecutable. El código ejecutable es el archivo final que la computadora puede ejecutar directamente. Estos conceptos son clave en el proceso de compilación, donde el código fuente se transforma en código ejecutable mediante herramientas como compiladores y enlazadores.

e) __Proceso de obtención de código intermedio para su 
ejecución en máquinas virtuales a partir del código fuente. Herramientas implicadas.__

El código intermedio es una representación del código fuente que puede ejecutarse en máquinas virtuales, como el bytecode de Java. Este proceso implica herramientas como editores para escribir el código, compiladores para traducirlo a código intermedio y máquinas virtuales para ejecutarlo. Este enfoque permite portabilidad, ya que el mismo código intermedio puede ejecutarse en diferentes plataformas sin modificaciones.

f) __Clasificación de los lenguajes de programación: Lenguajes compilados, interpretados y virtuales. Código máquina. Lenguajes de medio y alto nivel.__

Los lenguajes compilados, como C++, se traducen directamente a código máquina. Los interpretados, como Python, se ejecutan línea por línea mediante un intérprete. Los lenguajes virtuales, como Java, generan código intermedio para máquinas virtuales. El código máquina es el lenguaje binario que la CPU entiende.
